# Contexto da Feature: Chat de Consultoria Neural

## Visão Geral
Esta feature provê uma interface de chat interativo (estilo NotebookLM) para que o usuário possa interagir em tempo real com o documento analisado, tirando dúvidas, pedindo simulações de auditoria ou gerando textos derivados (e-mails, comunicados).

## Objetivos (Sessão 24)
- Testar duas abordagens de UX/UI: Painel Lateral (Slide-over) e Balão Flutuante (Floating Action).
- Garantir que a IA atue estritamente sob a "fonte da verdade" do documento carregado.
- Manter o padrão visual premium (glassmorphism, animações suaves).

## UX/UI
- **Painel Lateral**: Melhor para sessões longas de consultoria, preservando a visão do relatório.
- **Balão Flutuante**: Mais discreto, ideal para tiras-dúvidas rápidos sem obstruir a interface principal.

## Fluxo
- O chat é ativado apenas após a análise inicial ser concluída.
- A interface de chat deve gerenciar seu próprio estado de mensagens localmente antes da persistência no backend.
