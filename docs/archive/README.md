# Arquivo Historico

Esta pasta contem materiais descontinuados, analises antigas e artefatos legados.

1. Conteudo em `docs/archive/` nao e referencia oficial para decisao tecnica atual.

3. Documentos ativos devem ficar nas areas:
   - `docs/architecture/`
   - `docs/operations/`
   - `docs/reference/`
