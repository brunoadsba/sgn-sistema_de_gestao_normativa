# Contexto da Feature: Normas (Catálogo)

## Visão Geral
Catálogo de consulta das 38 Normas Regulamentadoras (NRs) brasileiras, com links oficiais e anexos.

## Decisões Arquiteturais e Insights (Sessão 21)
- **Busca via URL (nuqs)**: Implementada a biblioteca `nuqs` para gerenciar o estado do filtro de busca diretamente na URL (`?search=`). Isso permite que o estado da busca seja "copiável" e persistente a recarregamentos de página sem necessidade de redimensionar o `useState` no pai.
- **Data Fetching Server-Side**: `src/app/normas/page.tsx` agora é um Server Component que busca todas as normas via `getNormas()`. Essas normas são passadas como `normasIniciais` para o componente cliente enumerar e filtrar.
- **Desempenho**: A filtragem agora ocorre totalmente no cliente através do `useMemo` em `ListaNormasDinamica.tsx`, garantindo feedback instantâneo ao usuário enquanto digita (debounced em 300ms para atualização de URL).
- **Correção de UI**: O título principal foi ajustado para evitar o corte da letra 'g' em "Regulamentadoras", um bug comum detectado ao usar `bg-clip-text` com `tracking-tighter`.

## Links e Anexos
- Todas as NRs possuem `urlOficial` mapeada para o domínio da CTPP/MTE.
- Anexos específicos são mapeados em `urlAnexos`, permitindo navegação granular para tabelas e portarias complementares.
